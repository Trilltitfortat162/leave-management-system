\# 🚀 Leave Management System (Mini SaaS)



A production-ready Leave Management System built using Flask and SQLAlchemy with role-based access, workflow management, and approval system.



\---



\## 🔥 Features



\- 👤 User Authentication (Login/Register)

\- 🛡️ Role-Based Access (Admin \& Employee)

\- 📅 Apply Leave System

\- 🔄 Approval Workflow (Pending → Approved/Rejected)

\- 📊 Dashboard Analytics

\- 🔐 Secure Password Hashing

\- 💼 Business Logic Validation



\---



\## 🧑‍💻 Tech Stack



\- Python

\- Flask

\- SQLAlchemy

\- SQLite

\- Bootstrap 5



\---



\## 🚀 How to Run



git clone https://github.com/ReshmaPyan-2004/leave-management-system.git  

cd leave-management-system  

python app.py  



Open: http://localhost:5000
